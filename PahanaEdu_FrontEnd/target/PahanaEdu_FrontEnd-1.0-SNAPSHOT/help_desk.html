Help Desk

Overview: Displays total items, total revenue, and active total customers.

How to get there: Click the "Dashboard" button in the sidebar.

Tip: Use the quick view cards to monitor performance at a glance.

Overview: Add, view, update, or delete customer details.

How to get there: Use the sidebar menu "Customer Management".

Tip: Double-check contact numbers before saving. Editing requires selecting the customer from the list.

Overview: Manage book and stationery items (code, title, author, price, stock).

How to get there: Use the sidebar menu "Item Management".

Tip: A Unique item code is generating when you select the category. You can search for items using the filter bar.

Overview: Generate and view customer bills. Download or print as needed.

How to get there: Use the sidebar menu "Billing Management".

View All Bills: Click on green button "View All Bills" to view submitted bills and print them.

Print Button: Disabled after the first print to avoid duplicates. You can still view the bill in the downloads.

Delete Button: Permanently removes the bill (use with caution!).

Overview: Logs you out of the system and redirects to the login page.

Confirmation: A popup appears before logout to avoid accidental exit.

How to use: Click "Exit" in the sidebar. Confirm when prompted.
